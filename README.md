# Samsung-A03-CUSTOM-ROM
** test  and work on SM-A035F(A035FXXS9CYG1)

1.Extract the .7z to .tar

2.Use Odie to flash rom file in AP.

3.If it don't boot go to recovery mode and Wipe Data / Factory Reset.

4.To update the new version go to setting > Phh treble Settings > Updater > click Download and install.

5.Wait for complete download.

6.Restart and ENJOY.
